# dot-atom

## Install
This atom config is available as [Ellipsis][Ellipsis] package.

#### Ellipsis
``` shell
    ellipsis install kulynyak/dot-atom
```

## Maintain
To keep everything up-to-date;

#### Ellipsis
```shell
    ellipsis pull atom
```
